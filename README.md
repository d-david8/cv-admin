# Portfolio Admin Application

A web-based admin application built with **Laravel** and **Tailwind CSS** for managing professional profiles, experiences, education, projects, and tech stacks.

The application is designed to be clean, modular, and easy to extend, making it suitable for personal portfolios or professional CV management systems.
